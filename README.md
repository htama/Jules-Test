# テトリスゲーム

これはブラウザで動作するシンプルなテトリスゲームです。

## 利用方法

1.  このリポジトリのファイル一式（`index.html`, `style.css`, `script.js`）を同じフォルダにダウンロードします。
2.  `index.html` ファイルをウェブブラウザで開くとゲームが開始されます。

特別なソフトウェアのインストールは必要ありません。

## 操作説明

キーボードでテトリミノを操作します。

-   **← (左矢印キー)**: テトリミノを左に移動します。
-   **→ (右矢印キー)**: テトリミノを右に移動します。
-   **↓ (下矢印キー)**: テトリミノを下に速く落とします（ソフトドロップ）。
-   **↑ (上矢印キー) または スペースキー**: テトリミノを回転させます。

ゲームオーバーになった場合は、ブラウザのページを更新することで再挑戦できます。

楽しんでください！
